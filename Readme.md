# 衣寻 - 找到你的衣服

## 介绍

这个系统是一个基于 `微信小程序` (前端) 和 `flask` (后端)，用于寻找丢失的衣服（特别是校服）。

## 项目规划

- service: 后端
- - school: 学校相关接口
- - service: 服务相关接口
- gui: 前端
- - <i>暂时未制作</i>
- - maker: 制造商界面
- - school: 学校界面
- - user: 普通用户界面
